魔兽插件 · 国服正式服

一个斜杠命令，查到物品的英文名。

为中文客户端玩家准备的、单用途的魔兽插件。把物品链接、物品 ID 或中文名丢给 /en，英文名出现在一个可复制的窗口里。Sporefall 12.0.7 已适配。

三种查法

同一个斜杠命令，三种输入 — 你眼前是什么就用什么。

1

按物品链接

Shift 点一下物品，再敲 /en。链接会被自动解析成物品 ID。

2

按物品 ID

如果手头有 Wowhead 或其他来源的 ID，直接传：/en 19019。

3

按中文名

直接输入中文名，插件会反向查表。在和外语玩家交流、上 Wowhead 查信息时用。

安装

两个插件要一起装：主插件和数据子插件。

CurseForge
v2.3 · Sporefall 12.0.7
前往 CurseForge →
手动安装
把 CopyEnglishName 和 CopyEnglishName_Data 拖进 Interface/AddOns/。
前往仓库 →
数据子插件
CopyEnglishName_Data — 35 个按需加载的数据块。
前往数据仓库 →

数据子插件是必需的 — 它存着中英对照表。没有它 /en 没东西可查。

它能做什么、不做什么

能做

  • 查物品的英文名
  • 三种输入：物品链接、物品 ID、中文名
  • 在游戏内可复制的窗口里展示结果
  • 按需加载词典，内存占用低

不做的

  • 不自动扫描 tooltip
  • 不查技能
  • 不查 NPC 名
  • 不直接复制到系统剪贴板

单用途是设计上的选择。词典和 Raidpapa 浏览器扩展共用，物品名修正两处都会生效。
