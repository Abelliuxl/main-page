来自热门 licon/ez-translate 的轻量化 fork。划词翻译、右键图片直接问 AI、自带可配置 OpenAI / Anthropic 兼容端点 — 用你用得起的模型。

在 GitHub 上查看 → 添加到 Chrome

能做什么

不只能翻译，还有真正可用的 Ask AI，翻译帮不到的它来补。

01

划词翻译

选中文字后鼠标旁出现图标，点击就在原位出翻译。

02

弹窗翻译

点工具栏图标，输入或粘贴即可翻译；页面上选中的文字会自动填入。

03

可拖动弹窗

拖动翻译窗口到任意位置，想摆哪摆哪。

04

右键图片 Ask

右键任意图片 → "Ask AI" — 调你的 Vision LLM 出描述、OCR 或问答。

05

独立 Vision LLM

图片 Ask 用单独配置的模型，便宜的文本模型和贵一点的视觉模型可以共存。

06

100+ 语言

智能语种检测、自定义源语言与目标语言，覆盖 100+ 种。

07

一键复制

结果上无需额外按钮，文本可直接选中复制。

08

本地优先存储

API Key 默认存放在 chrome.storage.local，可选同步到已登录的浏览器。

可接的模型

常用的内置了，剩下的用自定义端点。

OpenRouter SiliconFlow Longcat AI MiniMax Zhipu AI Moonshot Kimi DeepSeek Qwen Doubao 自定义 OpenAI 兼容 自定义 Anthropic 兼容
